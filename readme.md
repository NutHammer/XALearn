# 学E西安自动刷课工具（西安干部网络学院自动刷课工具）
手动登录、选课后，本程序可以自动学完所有已选课程。  
可在播放异常时自动重试。  
使用selenium模拟点击实现，一次只能登录一个账号。    

# 兼容性

支持amd64处理器的windows和linux系统。  
支持arm64处理器的linux系统。    

# 使用发行版
 0.下载与你的处理器与操作系统对应的发行版并解压缩到非系统盘。
    
 1.安装chrome浏览器（arm架构的发行版不需要此步骤）。  
  官网下载地址https://www.google.cn/chrome/  
    
 2.运行程序，等待程序启动浏览器。  
  windows：运行main.exe    
  UOS：运行main  
    
 3.完成登录和选课。  
  
 4.切换回本软件窗口，输入要学习的课程序号，按回车。  
  
 5.等待程序运行。  
  如果出现打开新页面后视频没有开始播放，请修改浏览器设置为自动播放视频。
  程序开始显示播放进度后就可以开始进行其他工作，保持浏览器和本程序在后台运行即可（可以最小化，但不能同时运行大型游戏等需要持续保持前台的软件）。  
  程序运行时，如果需要使用浏览器，请使用其他浏览器，不要在播视频的这个浏览器上操作。  
  软件正常运行时，会每10秒输出一个句号，出现其他情况说明软件已停止运行。  

# 使用源码运行
1. 下载源码，根据操作系统与处理器，按照对应的requirement文件安装依赖。
2. windows系统跳过此步。linux系统将bin文件夹下的chromedriver_linux_处理器架构.py名称修改为chromedriver.py。
3. arm架构的linux系统下，下载firefox和对应驱动程序放置于res文件夹中。（发行版已打包进去，可以直接下载发行版从其中拷贝。）
4. 运行bin/main.py
5. 按照发行版使用步骤3-5执行。
